# ConnectionQuest

Teacher ranking app with a browser UI and a minimal Node.js server for deployment on platforms like Render.

## Local start

1. Install dependencies:

```bash
npm install
```

2. Start the server:

```bash
npm start
```

3. Open:

```text
http://localhost:3000
```

## Render settings

- Build Command: `npm install`
- Start Command: `npm start`
